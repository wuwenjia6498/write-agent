# 屏蔽词库

> 用于去除AI腔调，提升内容真实感。在 S8 审校步骤使用。

---

## 开场陈词滥调

| 禁用短语 | 原因 | 替换建议 |
|---------|------|---------|
| 在当今.*?时代 | 过于书面化，缺乏个性 | 现在 / 这些年 / 最近 |
| 随着.*?的发展 | 套话，无实际信息 | 直接进入主题，删除此类铺垫 |
| 众所周知 | 既然众所周知就不用说 | 删除，或改为"很多人可能不知道" |

## 逻辑结构词过度使用

| 禁用短语 | 原因 | 替换建议 |
|---------|------|---------|
| 首先.*?其次.*?最后 | 过于程式化，缺乏自然流畅感 | 用自然语言过渡，如"除此之外"、"还有个有意思的是" |
| 综上所述 | 典型AI腔，过于正式 | 所以 / 这么看来 / 说到底 |
| 总而言之 | 书面语过重 | 简单说 / 我的看法是 / 我想说的是 |
| 不仅.*?而且 | 过于书面 | 改为分句表达，更口语化 |

## 模糊表达

| 禁用短语 | 原因 | 替换建议 |
|---------|------|---------|
| 在一定程度上 | 废话，不传递信息 | 删除或具体说明程度 |
| 某种意义上 | 模糊无力 | 具体说是什么意义 |
| 有助于 | 模糊，不够直接 | 能 / 可以 / 会 |
| 相关.*?方面 | 废话连篇 | 具体说明是什么 |

## 冗余表达

| 禁用短语 | 原因 | 替换建议 |
|---------|------|---------|
| 进行.*?活动 | "进行"多余 | 直接用动词，如"开展活动"→"做活动" |
| 实现.*?目标 | "实现"多余 | 达到目标 / 完成目标 |
| 具有.*?特点 | 绕弯子 | 是 / 有 |

## 过度客观（缺乏态度）

| 禁用短语 | 原因 | 替换建议 |
|---------|------|---------|
| 据报道 | 过于新闻化，缺乏温度 | 我看到 / 有报道说 |
| 研究表明 | 学术腔过重 | 研究发现 / 有研究说 / 科学家们发现 |
| 值得注意的是 | 过于正式 | 有意思的是 / 你会发现 / 特别的是 |

## 过度热情（微商风）

| 禁用短语 | 原因 | 替换建议 |
|---------|------|---------|
| 亲 | 微商风格，不专业 | 删除 |

| 哦~ | 刻意卖萌 | 删除或改为正常表达 |

## 审校检查清单

- **句子长度**：检查是否有超过40字的长句，拆分为短句
- **段落长度**：每段不超过200字，保持节奏感
- **个人色彩**：是否融入个人观察、案例或素材？
- **具体例子**：是否有具体的例子而非泛泛而谈？
- **自然语调**：读起来是否像人在说话，而非机器输出？
- **情感共鸣**：是否有情感共鸣点，而非冷冰冰的陈述？
