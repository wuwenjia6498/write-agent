# ⚡ 快速启动指南

## 🎯 第一次启动项目？按照以下步骤操作：

### 步骤 1：安装前端依赖
```bash
npm install
```

### 步骤 2：安装后端依赖
```bash
cd server
python -m venv venv
venv\Scripts\activate  # Windows
pip install -r requirements.txt
```

### 步骤 3：配置环境变量（可选）
如果要完整测试AI功能，需要配置：
```bash
cd server
copy .env.example .env
# 然后编辑 .env 文件，填入你的 ANTHROPIC_API_KEY
```

### 步骤 4：启动后端服务
```bash
cd server
python main.py
```
✅ 后端服务运行在：http://localhost:8000

### 步骤 5：启动前端服务（新终端）
```bash
npm run dev
```
✅ 前端服务运行在：http://localhost:3000

## 🎨 访问页面

1. **主页**：http://localhost:3000
2. **频道管理**：http://localhost:3000/channels

4. **API文档**：http://localhost:8000/docs

## 📋 快速测试流程

1. 访问 http://localhost:3000
2. 点击"频道管理"查看三个频道配置
3. 点击"开始创作"进入工作台
4. 选择一个频道（如"深度阅读"）
5. 输入需求并启动工作流
